新建引用窗口在EndNote21中添加引用

当无法采用其他辅助手段导入，或引用非网络资源时，可手动键入引用的各项信息。
  1. 打开EndNote 21软件。
  2. 在All Reference界面上选择“创建引用”图标。窗口如下图所示

  3. 在新打开的参考文献窗口中，选择适当的参考文献类型（例如，Journal Artical、Book等）。
  4. 在相应的字段中输入参考文献的详细信息（如作者、标题、出版日期、DOI等）。
    注：
    1. 注意Author字段的英文名识别格式为Last Name, First Name或 First NameLast Name。请确保每行输入一个名称。
    2. 填充尽可能多的字段，至少要包含在论文参考文献列表中要体现的内容。
    3. 如果某些文字是首次出现在该字段中，则该部分内容将显示为红色。
  5. 如果需要添加其他字段，点击窗口底部的“+”按钮并选择相应字段。
  6. 在完成填写参考文献信息后，点击窗口右上角的save按钮以保存创建的引用。
  7. EndNote 21将自动将新添加的参考文献保存到Library中。

如何解决无法从外部数据库导入文献引用文件

出现条件

当尝试将参考文献从中国知网或其他数据库将引用文件导入EndNote 21时遇到问题。

问题原因

导入参考文献失败的可能原因：

  • Import Options设置不正确。
  • Text Translation设置不正确。

解决方案

针对上述原因，下面是一些解决方案
  • 检查导入选项Import Options

    Import Option中有很多的选项，如果是导入PDF，则选择PDF Fileor Folder，如果是网上引用模式下载的格式，则对应选择Endnote Import 或者ISI-CE（此针对是在谷歌学术，百度学术或者web of science 上引用的模式，导入后会填充文章标题，卷号，DOI等字段信息）

  • 检查导入选项Text Translation。

    使用macos文本编辑方式打开下载的文档，并将其另存为，在另存为窗口可以查看纯文本编码格式

    请据此在Text Translation一栏中选择正确的编码格式。

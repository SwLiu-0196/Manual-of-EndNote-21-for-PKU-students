从中国知网导入至EndNote 21

这里介绍的是从中国知网导入引用文件的方法步骤，也适用于其他用文件导入的情况，如遇问题请参照这里
  1. 登录北京大学图书馆网站进入中国知网。
    注： 请确保在校内连接或远程连接北京大学校园网的情况下打开知网，使用VPN可能导致无法登录知网北京大学图书馆账户。
  2. 在知网搜索框中输入相关关键词，找到并选择需要导入的文献。
  3. 在文献详细信息页面，查找并点击引用选项。
  4. 点击弹窗右下角更多引用格式，在左侧栏目中选择导出格式为EndNote。
  5. 点击上方导出下载引用文件
    注：

    中国知网通常提供txt格式的引用文件。

  6. 下载导出文件并将其保存在计算机上的一个易于访问的位置。
  7. 打开EndNote 21软件。
  8. 在EndNote中，选择文件(File)> 导入(Import) 。
  9. 浏览并选择刚刚下载的文献导出文件。
  10. 点击弹窗左下角显示选项(Show Options)后如下图所示。

    Import Options选择"Endnote Import"；

    Duplicates选择"Import All"；

    Text Translation选择"Unicode(UNTF-8)" (如何查看纯文本编码格式)。

  11. 确认导入设置，然后点击“确定”按钮完成导入过程。
  12. 现在，你的中国知网文献已成功导入到你的Library中，你可以在EndNote中管理和引用这些文献。

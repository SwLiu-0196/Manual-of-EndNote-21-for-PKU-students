下载与安装

前往北京大学正版软件共享平台官网http://software.pku.edu.cn，下载并安装 EndNote 21。

  1. 登陆网站，点击关闭网页提醒。
  2. 在网页右上角搜索框内输入软件名称找到 Mac 版本的 EndNote 21，下载。
  3. 若搜索结果中没有 21 版软件，请点此解决：找不到软件？
  4. 打开安装包，双击软件图标启动安装程序。
  5. 如下所示，依次点击新弹窗中的 Install > Next > Next > Done，完成安装。
  6. 安装完毕后，出现弹窗提醒软件更新（若无提醒请忽略），若无需更新，可点击 Ignore This 或 Cancel 回到主界面。若需要更新，请参见下一步。
  7. 依次点击新弹窗中的 Download and Install > Next > Next > Yes > Done，完成软件更新。
  8. 更新完成后，点击 Ignore This 或 Cancel 回到软件主界面。
